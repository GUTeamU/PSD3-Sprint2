Sprint2
=======

# Installation

1. Clone the repository

2. Run the `install.sh` script

3. Setup the environment:
```bash
source bootstrap.sh
```

4. Build the application:
```bash
mvn clean package
```

# Usage

1. Setup the environment:
```bash
source bootstrap.sh
```

2. Run the application:
```bash
java -jar target/gs-securing-web-0.1.0.jar
```
